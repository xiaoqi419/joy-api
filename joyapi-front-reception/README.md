# joyapi-front-reception

vite vue3 project

### Frameworks

- Vue 3.x
- Vite 4.x
- TypeScript 4.x
- TSX
- Vue Router
- Pinia
- Element Plus
- SVG Icon
- Axios
- SASS / SCSS
- UnoCSS
- ESLint Standard

## Project setup

```
pnpm install
```

### Compiles and hot-reloads for development

```
# using .env.dev profile by default
pnpm run dev
pnpm run dev:dev

# using .env.uat profile
pnpm run dev:uat

# using .env.prod profile
pnpm run dev:prod
```

### Compiles and minifies for production

```
# using .env.dev profile
pnpm run build:dev

# using .env.uat profile
pnpm run build:uat

# using .env.prod profile
pnpm run build:prod
```
